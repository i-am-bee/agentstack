## Agents

Agents made discoverable and composable through BeeAI.

| Name                                                                                                            | Framework | Description                                                                                                                                                 |
| --------------------------------------------------------------------------------------------------------------- | --------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [chat](https://github.com/i-am-bee/beeai-platform/tree/main/agents/official/beeai-framework/chat)               | BeeAI     | The agent is an AI-powered conversational system with memory, supporting real-time search, Wikipedia lookups, and weather updates through integrated tools. |
| [sequential_workflow](https://github.com/i-am-bee/beeai-platform/tree/main/agents/official/sequential-workflow) | -         | The agent orchestrates a sequence of text-processing AI agents, managing the flow of information and instructions between them.                             |

## CLI Usage

```sh
beeai run <agent_name>
```
