## Usage

```sh
beeai run AGENT_NAME
```

## Agents

The following is a list of the official agents:

| Name                                                                                                                                  | Framework | Description                                                                                                                                                                                                                                                                                 |
| ------------------------------------------------------------------------------------------------------------------------------------- | --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [agent-docs-creator](https://github.com/i-am-bee/beeai/blob/main/agents/official/agent-docs-creator)                                  | BeeAI     | The agent analyzes AI source code to generate structured, clear, and complete documentation in a consistent template, supporting multiple languages.                                                                                                                                        |
| [chat](https://github.com/i-am-bee/beeai/blob/main/agents/official/beeai-framework/src/chat)                                          | BeeAI     | The agent is an AI-powered conversational system with memory, supporting real-time search, Wikipedia lookups, and weather updates through integrated tools.                                                                                                                                 |
| [content-judge](https://github.com/i-am-bee/beeai/blob/main/agents/official/beeai-framework/src/content-judge)                        | BeeAI     | Evaluates multiple documents and agent-generated content based on correctness, depth, clarity, and relevance, selecting the highest-scoring one. It ensures optimal document quality for research, content validation, and knowledge refinement.                                            |
| [podcast-creator](https://github.com/i-am-bee/beeai/blob/main/agents/official/beeai-framework/src/podcast-creator)                    | BeeAI     | The agent creates structured podcast-style dialogues optimized for AI-driven text-to-speech (TTS). It formats natural conversations with a lead speaker and an inquisitive co-host, ensuring realistic interruptions and follow-ups. The output is structured for seamless TTS integration. |
| [sequential-workflow](https://github.com/i-am-bee/beeai/tree/main/agents/official/composition/src/composition/sequential_workflow.py) | -         | The agent orchestrates a sequence of text-processing AI agents, managing the flow of information and instructions between them.                                                                                                                                                             |
| [supervisor](https://github.com/i-am-bee/beeai/blob/main/agents/official/beeai-supervisor/src/supervisor/supervisor.ts)               | -         | The agent that autonomously breaks down tasks, assigns them to suitable agents, manages execution, evaluates outcomes, adapts workflows dynamically, and iterates until achieving an optimal solution.                                                                                      |
