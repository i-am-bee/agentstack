## Installation

1. **Install** BeeAI using [Homebrew](https://brew.sh/) (or see the [installation guide](https://docs.beeai.dev/get-started/installation) for other methods):

```sh
brew install i-am-bee/beeai/beeai
brew services start beeai
```

2. **Launch** the web interface:

```sh
beeai ui
```

3. **Use** from the terminal:

```sh
# List commands
beeai --help

# List all available agents
beeai list

# Run the chat agent
beeai run chat
```

## Configure API keys

After installation, configure your API keys to enable BeeAI to interact with your preferred service provider. For this example, we'll use Groq.

1. **Add environment variables** by running:

```bash
beeai env add LLM_MODEL=deepseek-r1-distill-llama-70b
beeai env add LLM_API_BASE=https://api.groq.com/openai/v1
beeai env add LLM_API_KEY=gsk_[...] <- your API key here
beeai env check
```

2. **Verify configuration** by running:

```bash
beeai env check
```
