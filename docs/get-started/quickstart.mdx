## What is BeeAI?

BeeAI is an open platform to help you discover, run, and compose AI agents from any framework and language. Whether building your agents or looking for powerful existing solutions, BeeAI makes it easy to find, connect, and orchestrate AI agents seamlessly.

## Get started

<Steps>
   <Step title="Install">

Install BeeAI using [Homebrew](https://brew.sh/) (or see the [installation guide](/get-started/installation) for other methods):

```sh
brew install i-am-bee/beeai/beeai
brew services start beeai
```

   </Step>
   <Step title="Configure LLM provider">

```sh
beeai env setup
```

   </Step>
   <Step title="Launch the web interface:">

```sh
beeai ui
```

   </Step>
   <Step title="Use from the terminal:">

```sh
# List commands
beeai --help

# List all available agents
beeai list

# Run the chat agent
beeai run chat
```

   </Step>
</Steps>
