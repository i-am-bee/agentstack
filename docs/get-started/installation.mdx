## Installation methods

BeeAI can be installed on Mac or Linux. We support multiple installation methods:

### Homebrew (Recommended)

A cross-platform package manager for Linux and Mac. This method will install all dependencies for you, as well as manage background services.

1. If you don't already use Homebrew, install it using the [official guide](https://brew.sh/).

2. Install **BeeAI**:
```bash
brew install i-am-bee/beeai/beeai && brew services start beeai`
```

3. _(Optional)_ Install [Arize Phoenix](https://phoenix.arize.com/):
```bash
brew install i-am-bee/beeai/arize-phoenix && brew services start arize-phoenix`
```

<Tip>  
    To update to the latest version, run `brew upgrade beeai`.<br/> 
    Agents are auto-updated.  
</Tip>  

### PyPI

A Python-native method, requiring more manual setup.

1. Install [`pipx`](https://github.com/pypa/pipx):
```bash
`sudo apt install pipx`   # Debian/Ubuntu
`sudo dnf install pipx`   # Fedora
`sudo pacman -S python-pipx`  # Arch
```

<Tip>
    Alternatively, you can use a virtual Python environment with `pip` or `uv`.
</Tip>

2. Install BeeAI:

```bash
pipx install beeai-cli
```

3. _(Optional)_ Install [Arize Phoenix](https://phoenix.arize.com/):

```bash
pipx install arize-phoenix
```

4. Install **agent runtimes** (optional but required for agent use):

| **Runtime**               | **Description**                                                                                  | **Installation**                                                                                   |
|---------------------------|--------------------------------------------------------------------------------------------------|----------------------------------------------------------------------------------------------------|
| **Python**                | BeeAI uses `uv` for environment management.                                                      | Install via your OS’s package manager or follow the [official guide](https://docs.astral.sh/uv/getting-started/installation/). |
| **JavaScript / TypeScript** | BeeAI uses `npx` (part of Node.js).                                                              | Install Node via your OS’s package manager or follow the [official guide](https://nodejs.org/en/download). |
| **Docker**                | Optional; no built-in agents require this. BeeAI supports the Docker CLI for agent containers.   | Install via [Rancher Desktop](https://rancherdesktop.io/) if needed.                             |

5. **Start BeeAI service** by running:
```bash
beeai serve
```

6. _(Optional)_ **Start Arize Pheonix service** by running:
```bash
PHOENIX_GRPC_PORT=8336 phoenix serve
```

<Tip>  
    To update to the latest version, run `pipx upgrade beeai`.<br/> 
    Agents are auto-updated.  
</Tip>

## Set up API keys

1. Choose a provider: 

<AccordionGroup>
    <Accordion title="OpenAI">
    ```bash
    beeai env add LLM_MODEL=gpt-4o
    beeai env add LLM_API_BASE=https://api.openai.com/v1
    beeai env add LLM_API_KEY=sk_\[...] \# <- your API key here
    beeai env check 
    ```
    </Accordion>

  <Accordion title="Anthropic">
    ```bash
    beeai env add LLM_MODEL=claude-3-7-sonnet-20250219
    beeai env add LLM_API_BASE=https://api.anthropic.com/v1/
    beeai env add LLM_API_KEY=[...] # <- your API key here
    beeai env check
    ```
  </Accordion>

  <Accordion title="Groq">
    ```bash
    beeai env add LLM_MODEL=deepseek-r1-distill-llama-70b
    beeai env add LLM_API_BASE=https://api.groq.com/openai/v1
    beeai env add LLM_API_KEY=gsk_[...] # <- your API key here
    beeai env check
    ```
  </Accordion>

  <Accordion title="Ollama">
    ```bash
    beeai env add LLM_MODEL=llama3.3
    beeai env add LLM_API_BASE=http://localhost:11434/v1
    beeai env add LLM_API_KEY=ollama
    beeai env check
    ```
  </Accordion>

  <Accordion title="OpenRouter">
    ```bash
    beeai env add LLM_MODEL=google/gemini-2.0-pro-exp-02-05:free
    beeai env add LLM_API_BASE=https://openrouter.ai/api/v1
    beeai env add LLM_API_KEY=sk-or-v1-[...] # <- your API key here
    beeai env check
    ```
  </Accordion>
</AccordionGroup>

2. Configure API keys by running:
```bash
beeai env add <name>=<value>
```

3. Verify providers are in the `ready` state by running:
```bash
beeai provider list
```

<Note>
    Allow some time for the agent list to populate.
</Note>

<Tip>
If there's something wrong, run `beeai env check` to troubleshoot.
</Tip>
