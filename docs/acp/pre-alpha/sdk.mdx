The Python and TypeSript SDK for the Agent Communication Protocol enables standardized integration among clients, agents, and tools.

## Features

- ACP Client and ACP Server implementations supporting:
    - **Agent** discovery, invocation, list change notifications
    - **Tool** discovery, invocation, list change notifications
    - All other features enabled by [MCP](https://modelcontextprotocol.io/docs/concepts/architecture)
- Transport implementations:
    - STDIO
    - SSE
