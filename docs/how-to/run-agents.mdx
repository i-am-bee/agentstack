Once you've discovered agents and configured your environment, running them is simple.

The graphical user interface (GUI) provides an intuitive way to launch and interact with agents. Launch the web interface with `beeai ui`.

<Info>
Not all agents support GUI interaction.
</Info>

## Quickstart Examples

<Accordion title="Chat Agent">
Start an interactive chat session:
```bash
beeai run chat
```

Or send a single message:
```bash
beeai run chat "Hello! How can you help me today?"
```
</Accordion>

<Accordion title="GPT Researcher">
```bash
beeai run gpt-researcher "Research the latest developments in quantum computing"
```
</Accordion>

<Accordion title="Aider - Code Assistant">
```bash
beeai run aider "
Review this Python function and suggest improvements:

def calculate_fibonacci(n):
    if n <= 1:
        return n
    return calculate_fibonacci(n-1) + calculate_fibonacci(n-2)
"
```
</Accordion>

<Accordion title="Prompt Chaining GPT Researcher -> Chat Agent">
```bash
beeai run gpt-researcher "Latest developments in renewable energy storage" | beeai run chat "Summarize this research in 3 key bullet points"
```
</Accordion>

## Input Methods

### Interactive Mode

```bash
beeai run chat
```

### Direct text Input

```bash
beeai run gpt-researcher "Research the latest developments in quantum computing"
```

### Multi-line Input

```bash
beeai run aider "
Review this Python function and suggest improvements:

def calculate_fibonacci(n):
    if n <= 1:
        return n
    return calculate_fibonacci(n-1) + calculate_fibonacci(n-2)
"
```

### File Input

```bash
beeai run chat < my_file.txt
```

**Pipe output from one agent to another:**
```bash
beeai run gpt-researcher "Latest developments in renewable energy storage" | beeai run chat "Summarize this research in 3 key bullet points"
```

## Working with Agent Output

### Save Output to Files

```bash
beeai run chat "Write a technical blog post about Docker" > ~/blog-post.md
```

### Append to Existing File

```bash
beeai run chat "Add a conclusion section" >> ~/blog-post.md
```
