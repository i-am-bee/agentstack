## Key features

| Feature | Description |
| :------ | :---------- |
| **Framework agnostic** | Connect agents built from **any framework** in **any language** through our unified interface. BeeAI handles all communication protocols and data transformations between different implementations. |
| **Agent composition** | Build **powerful workflows** by connecting specialized agents together. |
| **Agent discovery** | Browse our curated catalog of **ready-to-use agents**. Use integrated search and filter by provider to find exactly what you need. |
| **Developer tools** | Create custom agents with our **Python and TypeScript SDKs**. Focus on your agent's core logic while BeeAI framework handles the rest. |
