Follow these steps to install, configure, and start using BeeAI. This guide will help you set up the environment, launch the web interface, and interact with BeeAI through the terminal.

<Steps>
   <Step title="Install BeeAI:">

Install BeeAI using [Homebrew](https://brew.sh/) (or see the [installation guide](/introduction/installation) for other methods):

```sh
brew install i-am-bee/beeai/beeai
beeai platform start
```

   </Step>
   <Step title="Configure LLM provider:">

```sh
beeai env setup
```

   </Step>
   <Step title="Launch the web interface:">

```sh
beeai ui
```

   </Step>
   <Step title="Use from the terminal:">

```sh
# List commands
beeai --help

# List all available agents
beeai list

# Run the chat agent
beeai run chat

# Compose agents
beeai compose sequential
```

   </Step>
</Steps>
